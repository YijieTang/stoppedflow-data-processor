# Stopped-flow data processor
## Install python and packages
1. make sure python3 is installed
2. install required packages 
```pip install -r requirements.txt```
3. requirements.txt lists a <b>superset</b> of required packages, some may be unnecessary
## Run jupyter notebook on a Linux-like system
1. jupyter may ask to set a password when started for the first time
2. run jupyter server on a local machine
```jupyter notebook```
3. open browser and visit http://localhost:8888
